Introduction to Semantic HTML

Semantic HTML uses tags that describe the meaning of content, not just its appearance.

Why Use Semantic Tags?

  • Better accessibility
  • Improved SEO
  • Easier to maintain

Common Structural Tags

Header and Navigation

The <header> contains introductory content and <nav> holds navigation links.

Main Content

The <main> tag contains the primary content of the page.

Quick Tip

Use only one <main> element per page!

Articles and Sections

<article> represents independent content, while <section> groups related content.

Click to see more details

Articles can be syndicated independently, like blog posts or news stories.
